# comm
## display the unique lines in `file1`, the unique lines in `file2` and the lines found in both files respectively.
`comm file1 file2`

## display only the lines that were found in both files
`comm -12 file1 file2`







# diff

## context format
`diff -c fil1 file2`


## unified format
`diff -u fil1 file2`
